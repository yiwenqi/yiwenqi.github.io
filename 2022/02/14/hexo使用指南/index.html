Hexo 日常使用指南

简介

Hexo 是一个快速、简洁且高效的基于 Node.js 的静态博客框架，使用 Hexo，您可以使用 Markdown 等轻量级语言来撰写博客文章，并将其转换为静态页面进行展示，而不需要像传统博客系统那样依赖于数据库和服务器。Hexo 具有快速、简洁、高效等特点，非常适合个人博客、技术博客和文学博客等多种用途。

安装 Hexo

  1. 安装 Node.js：在 Node.js 官方网站下载并安装 Node.js。
